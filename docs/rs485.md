# 🏭 Industrial RS485 to MQTT Bridge

Selamat datang di panduan teknis **RS485 to MQTT Bridge** dari iothings.io. Modul ini dirancang untuk mengintegrasikan sensor industri (Modbus) ke sistem modern seperti Home Assistant atau Node-RED.

---

### 🔌 Skema Koneksi (Wiring)
* **A+ / B-:** Hubungkan ke bus RS485 perangkat sensor/meteran.
* **VCC:** Input 12V - 24V DC (Industrial Standard).
* **GND:** Grounding sistem.

---

### 🛠 Konfigurasi Modbus
* **Baudrate:** 9600 (Default) / 115200.
* **Data Bits:** 8 | **Stop Bits:** 1 | **Parity:** None.
* **Protocol:** Modbus RTU Over MQTT.

---

### 🚀 Integrasi Home Assistant
Gunakan konfigurasi `sensor` di YAML Home Assistant Bapak:

```yaml
sensor:
  - platform: mqtt
    name: "Industrial Energy Meter"
    state_topic: "iothings/rs485/data"
    unit_of_measurement: "kWh"
