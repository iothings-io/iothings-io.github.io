# 🛠 Panduan Setup Zigbee Gateway Pro

Selamat datang di panduan teknis **Zigbee Gateway Pro** dari iothings.io. Perangkat ini menggunakan modul **CC2652P/E18** dan **ESP32-S3**.

---

### 📋 Spesifikasi Hardware
* **Microcontroller:** ESP32-S3 (Dual Core, WiFi, Bluetooth).
* **Zigbee Module:** CC2652P (E18-MS1PA2) dengan antena eksternal.
* **Power Input:** 5V Micro USB / Terminal Block 12V-24V.
* **Compatibility:** Support Home Assistant (ZHA & Zigbee2MQTT).

---

### 🚀 Langkah Cepat Penggunaan
1.  **Powering:** Hubungkan perangkat ke sumber daya 5V atau 12V.
2.  **WiFi Setup:** Hubungkan ke hotspot "iothings-gateway" untuk konfigurasi WiFi.
3.  **Integration:** Masukkan alamat IP Gateway ke konfigurasi Zigbee2MQTT di Home Assistant.
